# 帮我用python写个脚本，文件名是deep_read.py

- 你需要使用到Playwright库。这个脚本不要引用其他的第三方的包
- 用系统自带的浏览器进行处理
- 这个脚本的运行参数我会用命令行传递给你，比如：python mvn_tool.py --HTML_PAGE "https://www.xxx.com/xxx" 
- 这个页面可能是需要登录之后才能阅读的，所以在链接打开之后，可能会跳转到登录页面。
  - 如果需要登录，那么你需要把页面打开，让用户自己输入身份信息进行鉴权，同时代码需要监听用户鉴权是否成功，如果超过1分钟没有监听到鉴权成功，你需要给出错误提示，终止后续的逻辑执行
  - 如果不需要登录，那么可以跳过这个步骤，也不需要打开任何窗口
- 当页面打开之后，你需要将当前页面的html完整的打印出来。
